    책자/제본 견적 안내

    물음표 버튼을 눌러 언제든 다시 확인할 수 있어요.

    • 접수 절차: 사양 선택 → 견적 확인 → 주문 접수 → 파일 업로드 → 검토 후 제작 진행
    • 사이즈: 규격 또는 완성 재단 사이즈(mm) 기준으로 선택해 주세요.
    • 페이지 수: 표지 포함 여부를 확인해 주세요. 중철은 4의 배수 페이지 권장.
    • 용지 선택: 표지는 내지보다 두꺼운 평량을 권장합니다.
    • 제본 방식: 페이지 수와 두께에 따라 선택 가능 방식이 달라질 수 있습니다.
    • 파일 기준: 300dpi, CMYK, 재단선 기준 작업 권장.
    사양이 애매한 경우 사용 목적과 희망 납기를 함께 기재해 주세요.
