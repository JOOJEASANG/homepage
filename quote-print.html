디지털 인쇄 견적 안내

물음표 버튼을 눌러 언제든 다시 확인할 수 있어요.

  • 접수 절차: 사양 선택 → 견적 확인 → 주문 접수 → 파일 업로드 → 검토 후 제작 진행
  • 사이즈: 규격 또는 완성 재단 사이즈(mm) 기준으로 입력해 주세요.
  • 인쇄면: 단면/양면 선택 시 데이터가 정확히 구분되어야 합니다.
  • 용지 선택: 평량이 높을수록 두꺼워집니다. 200g 이상은 오시 권장.
  • 파일 기준: 300dpi, CMYK, 재단선 기준 작업 권장.
사용 목적과 희망 납기를 함께 기재해 주시면 보다 정확한 안내가 가능합니다.
